# PriceX

## Description

PriceX is an AI-powered Dynamic Pricing & Marketing Automation Platform that leverages multimodal deep learning to predict optimal product valuations and generate high-converting ad creatives. By fusing visual data (images) with textual metadata, PriceX ensures accurate pricing while automating the creation of marketing content using Generative AI.

## Features

* AI Dynamic Pricing: Predicts fair market value using a custom model.
* Multimodal Analysis: Processes both product images and descriptions simultaneously to detect premium features and packaging nuances.
* Automated Ad Studio: Generates marketing hooks, Instagram captions, and email subjects using Google Gemini.
* Generative Visuals: Creates professional ad photography and lifestyle shots.
* Video Ad Generation: Animates static product images into short video clips.
* Campaign Dashboard: A centralized React interface for real-time pricing analysis and content management.
* History & Analytics: Tracks pricing trends and past predictions via a structured database.
